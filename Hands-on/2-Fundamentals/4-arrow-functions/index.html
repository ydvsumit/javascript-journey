Arrow Function

An arrow function is simply and special form of function expression that is shorter and therefore faster to write.

  • Syntax: const calcAge = (params) => {......}

    • => Difference between Arrow Function and Traditional Function (Function Declaration) :

    Feature

  • this Keyword
  • arguments object
  • constructor usage (new)
  • Hoisting

    • Arrow Function

  • Inherits from surrounding (lexical) scope
  • Not Available
  • Can't be used as a constructor
  • Not Hoisted

    • Traditional Function (Function Declaration)

  • Has it's own "this"
  • Available
  • can be used as a constructor
  • Hoisted
